# Architecture Decision Records

This directory holds Architecture Decision Records (ADRs): short documents
capturing a significant decision about the SDK's design, the context behind
it, and its consequences. They exist so future contributors and reviewers
share a reference instead of re-litigating settled questions.

Each ADR is numbered and immutable once accepted. If a decision changes, add a
new ADR that supersedes the old one rather than editing history.
